This is the official Node.js client for the **Elasticsearch Serverless** service. 

If you want to develop your JavaScript application with the Elasticsearch Stack, refer to the [Elasticsearch Client](https://github.com/elastic/elasticsearch-js) instead. 

If you want to develop your Node.js application with Elastic Enterprise Search, refer to the [Enterprise Search Client](https://github.com/elastic/enterprise-search-js/).
